# JDBC Driver Notes:
# ===================
# You need to download the appropriate JDBC drivers for your database:
#
# SQL Server:
#   Download: mssql-jdbc-*.jar from Microsoft
#   URL: https://docs.microsoft.com/en-us/sql/connect/jdbc/download-microsoft-jdbc-driver-for-sql-server
#
# MySQL:
#   Download: mysql-connector-j-*.jar from MySQL
#   URL: https://dev.mysql.com/downloads/connector/j/
#
# PostgreSQL:
#   Download: postgresql-*.jar from PostgreSQL
#   URL: https://jdbc.postgresql.org/download.html
#
# Oracle:
#   Download: ojdbc*.jar from Oracle
#   URL: https://www.oracle.com/database/technologies/appdev/jdbc-downloads.html
#
# Place the JAR files in a directory and specify the path with --jdbc-drivers-path
